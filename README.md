# Mini Solar System 🌌

A tiny Python + Pyglet demo for experimenting with motion.

## Setup

Install pyglet:

```
pip install pyglet
```

Run the program:

```
python starter_planets.py
```

## Your task

Open `main.py` and change the planet values:

* distance → orbit size
* speed → how fast it moves
* size → planet size

Make something weird 🚀


A simple orbit simulation becomes far more dynamic when gravity, velocity, and randomness are introduced—turning predictable motion into a living system where planets can wobble, collide, or spiral unpredictably, much like a real chaotic universe.


Challenge: Take this simulation further by exploring shader programs for visual effects, or add boundaries to see how planets react when they collide with limits of the system.

Projects will be judged for creativity and interactivity.
